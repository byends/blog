开发工具推荐-windows篇

#OTHERJun 20 2018

# 开发工具

  • PhpStorm - JetBrains 出品的 PHP IDE
  • Visual Studio Code - 微软出品的 IDE
  • Sublime Text - 文本编辑工具
  • Xshell - NetSarang 出品的 SSH 工具 （Free for Home/School）
  • Fiddler - Telerik 出品的抓包工具
  • Willow - 腾讯出的 Fiddler 插件，支持高效切换 HOST
  • SQLyog Ultimate - Webyog 出品的 MySQL GUI，支持 ssh 作跳板机
  • Terminus - 终端工具，支持 CMD、 PowerShell、Git-Bash
  • Enterprise Architect 8 - 时序图、流程图等各种图的制作工具
  • Axure RP 8 - 原型制作工具
  • Git - 版本控制工具

# 辅助工具

  • 2345 看图王 - 2345 出品的看图软件
  • Insomnia - A REST client ，API调试工具
  • Typora - Markdown 编辑器，完美的写作工具
  • CLCL - 剪贴板缓存工具
  • Listary - 高效的搜索启动工具

# Chrome 插件

  • Anything to QRcode - 任意网址或右键生成二维码
  • Axure RP Extension for Chrome - 使用 chrome 浏览本地原型图
  • EditThisCookie - Cookie 修改器
  • JSON Formatter - JSON 格式化工具（需要有正确的 Content-Type: application/json ）
  • Chrome Logger - 配合服务端的调试工具，可将日志展示在控制台
  • Octotree - Github 代码树
  • WEB前端助手(FeHelper) - 百度出品的前端助手
  • 新浪微博图床
  • Tampermonkey - 油猴（功能你懂的）
  • AdBlock - 屏蔽 AD
